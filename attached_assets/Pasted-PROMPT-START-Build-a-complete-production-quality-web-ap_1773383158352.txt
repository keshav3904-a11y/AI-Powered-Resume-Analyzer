PROMPT START

Build a complete production-quality web application called AI-Powered Resume Analyzer using Python + Streamlit + NLP + Machine Learning.

The application must follow a clean modular architecture, intuitive UI, and extremely easy navigation so that even non-technical users can operate it easily.

The system must analyze a resume uploaded by the user and calculate how well it matches a job role using AI-based logic.

1 PROJECT OBJECTIVE

Create an intelligent resume analysis system that:

• Accepts resumes in PDF format
• Extracts structured information from resumes
• Extracts skills, education, experience, projects, and certifications
• Allows user to select a predefined job role
• Compares resume with job role requirements
• Calculates match score (%)
• Shows missing skills
• Shows strength areas
• Shows visual analytics

This system should simulate an AI hiring assistant used by recruiters.
